HTML BASICS

Tag Description
<h1><h6> HTML can use six different levels of heading elements. The heading elements are ordered from the highest level <h1> to the lowest level <h6>.
<a> The anchor element is used to create hyperlinks in an HTML document. The hyperlinks can point to other webpages (using a full URL), files on the same server (using a relative path), or a location on the same page (using # followed by the id of the target element). The href value is the target of the link. <a> elements can be wrapped around an <img> element to turn an image into a link.
target The target attribute on an <a> element specifies where a hyperlink should be opened. A target value of "_blank" will tell the browser to open the hyperlink in a new tab in modern browsers, or in a new window in older browsers or if the browser has had settings changed to open hyperlinks in a new window.
<div> The <div> Element is used as a container that divides an HTML document into sections and is short for “division”. <div> elements can contain flow content such as headings, paragraphs, links, images, etc.
<p> The <p> paragraph element contains and displays a block of text.
<span> The <span> element is an inline container for text and can be used to group text for styling purposes. It can be used inside a <p> element to add a specific style to a section of the text.
<em> The <em> emphasis element emphasizes text and browsers will usually italicize the emphasized text by default.
<strong> The <strong> element highlights important, serious, or urgent text and browsers will normally render this highlighted text in bold by default.
<br> The <br> line break element will create a line break in text and is especially useful where a division of text is required, like in a postal address. The line break element requires only an opening tag and must not have a closing tag.
<ul> The <ul> unordered list element is used to create a list of items in no particular order. Each individual list item will have a bullet point by default.
<ol> The <ol> ordered list element creates a list of items in sequential order. Each list item appears numbered by default.
<li> The <li> list item element creates list items inside:
  • Ordered lists <ol>
  • Unordered lists <ul>
<img> HTML <img> elements embed images in documents. The src attribute contains the image URL and is mandatory. <img> is an empty element meaning it should not have a closing tag.
alt An <img> element can have alternative text via the alt attribute. The alternative text will be displayed if an image fails to render due to an incorrect URL, if the image format is not supported by the browser, if the image is blocked from being displayed, or if the image has not been received from the URL. The text will be read aloud if screen reading software is used and helps support visually impaired users by providing a text descriptor for the image content on a webpage.

SEMANTIC HTML

Tag Description
<header> Specifies a header for a document or section.
<nav> Defines a group of navigation links.
<main> Specifies the main content of a document.
<footer> Defines a footer for a document or section.
<section> Defines a section in a document.
<article> Defines independent, self-contained content which makes sense on its own.
<aside> Defines content aside from the page content.
<figure> Specifies self-contained content, like illustrations, diagrams, photos, etc. Will usually contain <img> and <figcaption> elements.
<figcaption> Defines a caption for a <figure> element. Usually used within the <figure> element.
<audio> Allows us to add audio to our website with src attribute. Adding the controls attribute will display audio controls.
<video> Allows us to add videos to our website with src attribute. Adding the controls attribute will display video controls.
<embed> Can be used to implement any type of media. Self-closing.
<mark> Defines marked/highlighted text.
<time> Defines a date/time.
<details> Defines additional details that the user can view or hide.
<summary> Defines a visible heading for a <details> element.

HTML TABLES

Table Tags

Tag Name Description
<table> Table Element has content that is used to represent a two-dimensional table made of rows and columns.
<thead> Table Head Defines the headings of table columns encapsulated in table rows.
<tbody> Table Body A semantic element that will contain all table data other than table heading and table footer content.
<tfoot> Table Footer Uses table rows to give footer content or to summarize content at the end of a table.
<th> Table Heading Used to add titles to rows and columns of a table and must be enclosed in a table row element.
<tr> Table Row Used to add rows to a table before adding table data and table headings.
<td> Table Data Can be nested inside a table row element to add a cell of data to a table.

Table Attributes

Attribute Name Description
colspan Column Span Indicates how many columns that particular cell should span within the table.
rowspan Row Span Indicates how many rows that particular cell should span within the table.
